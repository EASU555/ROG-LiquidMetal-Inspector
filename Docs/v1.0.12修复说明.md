# v1.0.12 修复说明

- 修复双烤期间 CPU 达到安全温度却不会停止的问题；CPU/GPU 安全计时改为真实墙钟时间。
- 修复 GPU 初始化失败或用户中止后压力任务可能残留的问题，并监视 OpenCL 调度是否仍在增长。
- GPU 压力建立必须同时满足负载与最低功耗；持续掉载会中止，低负载结果不允许显示正常。
- 修复平均 CPU 频率被重复除法导致显示约 85–140MHz 的问题。
- Core Ultra 混合架构改为 P-Core、E-Core 分组计算温差，避免跨架构误报；本机应显示 P 8 / E 16。
- 流程、持续异常和分析窗口全部按采样时间戳计算，不再把“样本数”当“秒数”。
- 增加 CPU/GPU 有效样本率门槛；用户停止、安全停止、传感器失败或压力失败都不能输出绿色结论。
- CSV、JSON、HTML 报告增加运行完成状态、P/E 核字段、有效率、分析时长和版本号；缺失字段显示为“未读取”。
- 默认规则与 G815LR CPU/GPU 档案真正参与运行和分析；CPU 与 GPU 测试只显示各自相关档案。
- 程序与文件版本统一为 1.0.12.0，增加 Per-Monitor V2 DPI 声明，并保持默认管理员权限。
- 新增 6 项自动化测试，覆盖混合核心、真实持续时间、中止状态、GPU 低负载、样本覆盖率和报告字段。
